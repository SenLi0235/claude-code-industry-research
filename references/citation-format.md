# 引用格式

## 正文中

```
...市场规模约 1234 亿元 [3]
...根据 Frost & Sullivan 报告，2024 年国内光伏装机量达 216GW [15]
...该技术路线的量产良率尚未突破 80% [UNSOURCED]
```

## 文末"参考资料"节（附录 C）

```
[1] 机构名《报告标题》, 年份, 页码/章节, URL（如有）
[2] 公司年报《XXX 2024 年报》, 第 X 页, 链接
[3] 国家统计局《XXX 行业统计公报》, 2024 年发布, 链接
[4] McKinsey & Company, "Global Energy Perspective 2024", 2024, p.45, https://...
```

每条引用必须包含：**机构 + 文件名 + 时间 + 可访问定位（页码或链接）**。
达不到上述四项的 → 归入附录 A `[UNSOURCED]`。

## 研究笔记 §C 登记格式（Bash >> 追加）

```
[N] 机构名《文件标题》年份, URL, 采纳数值=X
```

示例：
```
[15] Frost & Sullivan《中国光伏行业白皮书》2024, https://xxx, 2024年装机=216GW
[16] 国家能源局《2024年光伏发电建设运行情况》2025, https://xxx, 新增装机=120GW
```
