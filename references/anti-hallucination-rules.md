# 反幻觉硬规则

> 所有板块 subagent 和主 agent 共同遵守。

| 规则 | 做 | 不做 |
|---|---|---|
| 数字 | 必有引用或公式。无源标 `[UNSOURCED]` | 不写"约数千亿"无来源 |
| 公司案例 | 必有出处（年报/招股书/权威报道） | 不凭印象写"听说 XX 公司..." |
| 政策 | 必带文号、发文日期、链接 | 不写"国家鼓励 XX 产业" |
| 国外对标 | 必有 WebSearch 检索溯源 | 不假设"美国应该也这样" |
| 排名/市占率 | 必带年份和统计机构 | 不用"行业领先者"代替具体名次 |
| 趋势词 | 用"过去 3 年 CAGR=X%" | 不写"快速增长""蓬勃发展" |

## 当某项数据找不到可靠来源时

处理优先级：
1. **测算**（写出公式和假设，标"测算值"）
2. **标记 `[UNSOURCED]`**（继续走，不阻塞）
3. **记入附录 B**（标"待用户补充"）
4. **绝不允许**：编造一个看起来合理的数字

## `[UNSOURCED]` 使用规则

- 板块内 `[UNSOURCED]` 占比 ≤30% → 正常通过，附录 A 汇总
- 板块内 `[UNSOURCED]` 占比 >30% → 退回重写（检索深度不够）
- 禁止用"约/大约/据估计/据推测"模糊词代替 `[UNSOURCED]`

## 数据矛盾处理

两个来源打架时：
1. 先检查口径是否一致（广义 vs 中义 vs 狭义）
2. 口径一致但数值冲突 → 两个来源都列出来，注明差异和可能原因
3. 给出本报告采用值及理由
4. 记入笔记 §A
