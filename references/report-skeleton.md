# 输出报告骨架

```markdown
# <行业名> 行业研究报告

> 报告日期：YYYY-MM-DD｜研究目的：xxx｜口径：申万二级 XXX

## 核心结论
> Window C 由主 agent 撰写，≤800 字，3~5 条核心发现。让读者 1 分钟看懂"这个行业能不能进"。

## 一、行业定义与分类
## 二、行业生命周期阶段定位
## 三、产业链与价值链
## 四、竞争格局
## 五、市场规模与增长驱动
## 六、外部因素（PEST）
## 七、景气度指标体系

---
## 附录 A：数据矛盾与不确定性
## 附录 B：待补充资料清单（请用户手查）
## 附录 C：参考资料
```

## 附录格式

### 附录 A：数据矛盾与不确定性

| # | 指标/议题 | 来源 A（含数值） | 来源 B（含数值） | 本报告采用 | 选用理由 / 备注 |
|---|---|---|---|---|---|
| | | | | | |

含 `[UNSOURCED]` 来源不完整标记：只有 A 列填，B 列写"缺页码/缺年份/仅转载未溯源"。

### 附录 B：待补充资料清单

| # | 需要的数据 | 建议查询渠道 | 优先级（高/中/低） | 对哪个板块的结论影响 |
|---|---|---|---|---|
| | | | | |

### 附录 C：参考资料

```
[1] 机构名《报告标题》, 年份, 页码/章节, URL
[2] 公司年报《XXX 2024 年报》, 第 X 页, 链接
[3] 国家统计局《XXX 行业统计公报》, 2024 年发布, 链接
```

每条引用必须包含：**机构 + 文件名 + 时间 + 可访问定位（页码或链接）**。
达不到上述四项的 → 归入附录 A `[UNSOURCED]`。
