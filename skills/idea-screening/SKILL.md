# 投资标的筛选（可选模块 — Idea Screening）

> 行业研究报告回答"行业能不能进"，本模块进一步回答"进了之后重点关注谁"。
> 在行业研究报告完成后，用户可选用 `/screen-ideas` 命令触发本模块。

## 触发条件

用户说"筛选标的"、"有哪些标的"、"screen"、"idea generation"、"找标的"、"推荐公司"时触发。

## 工作流

### Step 1：确认筛选参数（AskUserQuestion 一次问完）

- **方向**：做多/做空/两者
- **市值范围**：大盘/中盘/小盘/不限
- **风格**：价值/成长/质量/特殊事件/不限
- **地域**：A 股/港股/美股/全球
- **数量**：输出 5-10 个标的

### Step 2：量化筛选

基于行业研究报告中 §2.2 的代表企业和 §2.3 的竞品池，运行对应风格的筛选：

**价值型**：P/E 低于行业中位数、EV/EBITDA 低于历史均值、FCF 收益率 >5%、破净、内幕增持

**成长型**：营收增速 >15%、利润增速 >20%、增速加速度为正、利润率在扩张、ROIC>15%

**质量型**：5 年+稳定增长、ROE>15%、低负债率、高 FCF 转化率

**做空型**：营收下滑或减速、利润率压缩、应收账款/库存异常上升 vs 营收、内幕减持、高估值无基本面支撑

**特殊事件型**：近 12 月 IPO/SPAC 锁定期到期、spin-off、重组出清、激进投资者介入、管理层变更

### Step 3：主题映射

对于主题性行研，额外做：
1. 映射行业价值链——谁直接受益 vs 间接受益？
2. 区分纯 play vs 多元化暴露
3. 评估哪些名字已被市场定价 vs 被低估
4. 找二阶受益者（市场尚未关联到主题的公司）

### Step 4：标的呈现

每个入选标的输出一页摘要：

```
### [公司名称] — [做多/做空] — [一句话论点]

| 指标 | 数值 | vs 同业中位数 |
|---|---|---|
| 市值 | | |
| EV/EBITDA (NTM) | | |
| P/E (NTM) | | |
| 营收增速 | | |
| EBITDA 利润率 | | |
| FCF 收益率 | | |
| ROE | | |

**论点（3-5 条）：**
- 为什么被错价
- 市场忽略/误解了什么
- 催化事件

**关键风险：**
- 什么会让论点失效

**建议下一步：**
- 深度模型？尽调？专家 call？
```

### Step 5：输出

- 短名单 5-10 个标的 + 一页摘要
- 筛选标准和方法论说明
- 跨标的对比总表
- 优先级排序：哪些标的应先做深度研究

## 重要原则

- 筛选是起点，不是终点——每个输出都需要基本面验证
- 最好的 idea 往往来自交叉点（如价值价位的质量公司）
- 避免拥挤交易——检查机构持仓、空头持仓、分析师覆盖密度
- 逆向标的必须有催化事件——没有催化的逆向就是长期被套
- 跟踪命中率——长期看哪些筛选方法产出最好
