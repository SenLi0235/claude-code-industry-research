# 景气度指标体系（§2.6）

基于 §2.2 产业链分析确定的"核心驱动环节"，建立高频跟踪体系。

## §2.6.1 纵向指标（产业链核心环节）

本行业产业链最敏感的供需信号在哪一环？列出 2-3 个高频先行指标：
- 如新能源车产业链：车企月销量、电池排产数据
- 如半导体产业链：设备出货量、晶圆厂开工率

## §2.6.2 横向指标（行业自有）

产销量、关键原材料价格、库存水平、开工率、PMI 子项、订单数据、运价指数等。

每个指标列出：

| 指标名 | 更新频率 | 数据源 | 当前最新值 | 同比 | 环比 |
|---|---|---|---|---|---|
| | | | | | |

## §2.6.3 未来 12-24 个月重点监测清单

从上述指标中精选 6-10 个对行业景气拐点最敏感的指标，一句话说明盯什么信号。

格式：
```
| 指标 | 盯什么 | 预警阈值 |
|---|---|---|
| XX月销量 | 连续3月低于XX万 → 需求拐点 | <XX万 |
| YY开工率 | 跌破XX% → 产能过剩信号 | <XX% |
```

## 对齐要求

景气度指标要与 §2.4 的增长驱动因子呼应：指标应覆盖主要的增长因子。
